== Description ==

The "telegram notifier" plugin is a powerful tool that helps you quickly process requests and respond to orders. In this way, you ensure a high level of service and convenience for your audience.

**Main features:**

1. **Automatic extraction of data from forms:** The plugin dynamically scans all your forms on the site and extracts the data entered by your user.
2. **Customized data selection:** You can choose exactly what information you want to receive from each form. The plugin settings allow you to select specific fields to monitor.
3. **Price and product names:** If your forms include orders for goods or services, the plugin automatically extracts the prices and product names and includes them in the message if you want.
4. **Real Time Telegram Notifications:** Every new message, order or request that comes from your website is immediately sent to you on Telegram. This allows you to quickly respond to important events.
5. **Compact and clear messages**: Messages in Telegram are structured and concise, allowing you to quickly understand the situation.


== Installation ==

1. Install the plugin through the WordPress plugins screen directly,
2. Activate the plugin through the 'Plugins' screen in WordPress
3. You will find all information about plugin settings in Admin sidebar menu -> Telegram Notifier
